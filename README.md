## Instalação
```bash
# Clone o Repositório:
$ git clone https://github.com/lucascantarelli/portainer.git

# Entre na pasta:
$ cd portainer

# De permissão de execução ao script:
$ sudo chmod +x build.sh

# Execute o script:
$ sudo ./build.sh
```
